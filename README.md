# Dragon Slayer

An Elder Scrolls Online addon that helps coordinate dragon hunts by sending randomized zone chat messages calling for help against dragons.

## Features

- Sends randomized messages to zone chat announcing a dragon sighting on the north side
- Includes 26 different message variations for variety
- Simple keybind activation

## Installation

1. Download the addon files
2. Extract the `DragonSlayer` folder into your ESO AddOns directory:
   - `Documents\Elder Scrolls Online\live\AddOns\DragonSlayer`
3. Restart ESO or reload the UI with `/reloadui`

## Usage

- Bind a key to "Send Dragon Slayer Call" in the ESO keybindings menu (under General > Dragon Slayer)
- When you spot a dragon, press the bound key to send a random call-to-arms message to zone chat
- This helps coordinate group efforts against dragons in the game

## Keybinds

- **Send Dragon Slayer Call**: Triggers a random zone chat message
